# Color Chai

Tiny Discord bot that generates swatches from color codes. Supports hex, rgb(a) and hsl(a) CSS codes.

## Setup

```bash
$ git clone https://github.com/ZaneH/color-chai.git
$ cd color-chai
$ mix deps.get
$ source .env # be sure to set DISCORD_TOKEN
$ mix run --no-halt
```
